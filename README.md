# 🛡️ منظومة تحليل التهديدات الذكية ومكافح الاحتيال المتكامل (Smart Phishing Analyzer Pro)

منظومة برمجية أمنية متكاملة ومطورة بالكامل بلغة **Python** وإطار عمل **Streamlit** للواجهات الرسومية.
تعتمد المنظومة على تقنيات الفحص الهجين وثلاث طبقات حماية متزامنة ومحصنة لصد هجمات الهندسة الاجتماعية والتصيد (Phishing).
تم تزويد التطبيق بآليات تتبع ذكية للروابط المخفية والمختصرة، بالإضافة إلى قدرات معالجة وتدفق الملفات النصية مباشرة.

---

## 🏛️ البنية الهندسية الشاملة وطبقات الفحص المتكاملة

تتبع المنظومة نموذج الدفاع السيبراني متعدد الطبقات لضمان دقة التحليل وتقليل نسب الإنذارات الكاذبة (False Positives):

### 1. طبقات الفحص الأساسية في المنظومة
* **طبقة الفحص المحلي الفوري (Local Indicator Layer):** تعتمد على مكتبة التعبيرات النمطية المنتظمة `re` لاقتناص الروابط والنطاقات وفحص الكلمات الدلالية للهندسة الاجتماعية المخزنة محلياً.
* **طبقة السمعة الأمنية العالمية (Global Threat Intelligence Layer):** تتصل برمجياً بـ `VirusTotal API v3` عبر مكتبة `requests` لإخضاع النطاقات المكتشفة لفحص أمني فوري عبر المحركات العالمية ومخرجات الـ `JSON`.
* **طبقة التحليل السياقي بالذكاء الاصطناعي (Advanced Contextual AI Layer):** تستخدم حزمة التطوير البرمجية الحديثة `google-genai` ونموذج `gemini-2.5-flash` لقراءة النوايا البشرية المبتكرة ومكافحة هجمات الـ Zero-Day Phishing.

## 🧪 تفكيك وشرح التقنيات والآليات المبتكرة والمتقدمة في الكود

يتضمن المشروع حزمة من الحلول البرمجية الذكية التي ترفع من كفاءته وقيمته التنافسية في التقييم البرمجي:

### أولاً: تتبع الروابط متعدد المستويات (Multi-Hop Redirection Tracking) - [ميزة محدثة]
* يستخدم المهاجمون حيلة ذكية عبر إخفاء الرابط الخبيث وراء سلسلة متتالية من الروابط المختصرة للالتفاف على أنظمة الفحص التقليدية.
* لحل هذه المعضلة، تم تطوير آلية الفحص في دالة `unshorten_url` لتعمل عبر حلقة تكرارية ذكية (`while loop / for loop`) تتتبع قفزات إعادة التوجيه لـ 5 مستويات متتالية.
* تعتمد الدالة على بروتوكول طلبات الترويسة الخفيفة `requests.head` مع فحص الكود المشفر للاستجابة وهيدر الـ `Location` دون تحميل المحتوى الكلي، مما يضمن كشف الوجهة الحقيقية النهائية للموقع بكل سرعة وأمان وبدون استهلاك موارد البيانات.

### ثانياً: نظام القائمة البيضاء المحلية الذكي (Whitelisting Mechanism) - [ميزة محدثة]
* لحماية خوادم التطبيق من استهلاك الحصص المجانية للـ APIs ولتجنب وقوع الذكاء الاصطناعي في فخ الإنذارات الكاذبة (False Positives)، تم ابتكار مصفوفة فحص استباقية تسمى `GLOBAL_WHITELIST`.
* تحتوي هذه المصفوفة على النطاقات العالمية الكبرى والموثوقة (مثل Google, Microsoft, Apple, GitHub).
* يقوم النظام بفصل النطاق وفحصه؛ وإذا ثبت أنه ينتمي للقائمة البيضاء، يتم تخطي الطلبات الخارجية للـ APIs وفحص الـ LLM فوراً، ومنح الرابط تقييم الأمان الكامل لتوفير وقت المعالجة وتأمين تجربة مستخدم سريعة جداً.

### ثالثاً: نظام التعلم المستمر المستدام ومحاكاة الآلة (Dynamic Knowledge Base Updating)
* تم ابتكار آلية متميزة تسمح للتطبيق بتطوير نفسه ذاتياً من خلال ملف تخزين محلي يسمى `knowledge_base.json`.
* عندما يكتشف نموذج Gemini كلمات احتيالية أو تعبيرات هندسة اجتماعية جديدة ومبتكرة، يقوم بإدراجها في مخرجات الجيسون.
* تلتقط الدالة البرمجية هذه الكلمات المكتشفة وتدمجها مع قائمة الكلمات المفتاحية القديمة المرفوعة في الذاكرة.
* يتم استخدام خاصية التحويل إلى مجموعات فريدة `list(set(keywords))` لضمان تنقية الكلمات تماماً من التكرار العشوائي.
* تُحفظ البيانات الجديدة فوراً بصيغة ترميز `encoding="utf-8"` ليدعم التطبيق الكلمات العربية والإنجليزية على حد سواء في الفحوصات القادمة.

### رابعاً: معالجة وقراءة تدفق الملفات المباشرة (File Stream Handling)
* يتيح التطبيق تجربة مستخدم مرنة عبر توفير أداة `st.file_uploader` المخصصة لاستقبال الملفات البريدية والنصية.
* يمكن للمستخدم رفع ملف البريد الإلكتروني الحقيقي بامتداد `.txt` أو `.eml` مباشرة عوضاً عن عمليات النسخ واللصق اليدوية الطويلة.
* يقوم التطبيق بقراءة دفق البيانات ثنائية الأبعاد (Binary Streams) وتحويلها إلى نصوص مقروءة باستخدام تابع فك التشفير `.decode("utf-8")`.
* يتم وضع النص المستخلص تلقائياً كـ (Default Value) داخل صندوق النصوص الرئيسي، مما يسمح بمراجعته وتعديله قبل بدء الفحص.

### خامساً: واجهة الوصول السريع والمساعدة الصوتية واللغوية (Accessibility & Assistance Hub)
* تم دمج بيئة برمجية تفاعلية لمساعدة ذوي الهمم وضعاف البصر عبر حقن أكواد جافا سكريبت مخصصة للاتصال بـ `Web Speech API`.
* عند الضغط على زر الاستماع الصوتي، يتم إلغاء أي عمليات نطق معلقة عبر الأمر `window.speechSynthesis.cancel()`.
* يتم إنشاء كائن نطق جديد وتحديد لغته لتكون العربية الفصحى المعتمدة `ar-SA` لقراءة تقرير السلامة وحيثيات الحكم الأمنية.
* تم توفير أداة ترجمة فورية متكاملة تقوم بأخذ النصوص وتحويلها للغة الإنجليزية عبر برومبت ذكاء اصطناعي مخصص لترجمة المصطلحات السيبرانية بدقة.

### سادساً: تحويل النظام إلى تطبيق مستقل قابل للتثبيت (PWA Transformation)
* تم حقن كود مخصص ومطور بلغة JavaScript داخل دالة المكونات الفرعية لـ Streamlit لتوليد متطلبات الـ PWA.
* يقوم الكود بإنشاء ملف تعريف افتراضي ديناميكي (Web App Manifest) يحتوي على اسم التطبيق، الألوان المعتمدة، وطريقة العرض Standalone.
* يتم تسجيل وتفعيل ملف (Service Worker) مشفر ومبني بترميز Base64 ليعمل المتصفح على إتاحة خيار تثبيت الموقع كتطبيق هاتف أو حاسوب مستقل تماماً.

## 📊 منطق احتساب مؤشر السلامة الرقمية (Safety Score Algorithm)

تم تصميم آلية الحساب البرمجي لتعبر عن نسبة "أمان وسلامة الرابط" بشكل عكسي وواضح للمستخدم النهائي:
1. **حالة الخطر المؤكد (Dangerous):** إذا أكد فحص السمعة العالمي أو محرك الذكاء الاصطناعي وجود تهديد، تنخفض النسبة مباشرة إلى **3%**.
2. **حالة الاشتباه (Suspicious):** في حال وجود لغة إلحاح وتخويف مع عدم وجود أدلة حظر قطعية، تُثبت النسبة عند **50%** (منطقة رمادية للحذر).
3. **حالة الأمان الكامل (Safe):** إذا خلت الرسالة من الكلمات الاحتيالية وجاء تقرير السمعة والذكاء الاصطناعي سليماً (أو إذا كان الرابط في القائمة البيضاء)، يمنح التطبيق نسبة **98%**.

## 🛠️ متطلبات التشغيل والتثبيت (Installation & Running)

لتشغيل هذا التطبيق على بيئتك المحلية، يرجى تتبع الخطوات التالية:
1. قم بتثبيت المكتبات الأساسية عبر ملف المتطلبات:
   `pip install -r requirements.txt`
2. قم بضبط مفاتيح الواجهات البرمجية كمتغيرات بيئة داخل جهازك لحمايتها:
   * `VIRUSTOTAL_API_KEY`
   * `GEMINI_API_KEY`
3. قم بتشغيل السيرفر المحلي للتطبيق عبر الأمر التالي:
   `streamlit run app.py`
